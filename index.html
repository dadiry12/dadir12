2. Enkle funksjoner

Etter å ha gitt en dag en ide om hva en funksjon er, skal vi se på et helt enkelt eksempel.

Vi difinerer her en funksjon med navn [skrivUtHilsen], som skriver ut medlingen "hei!" i en meldingsboks.

  function skrivUtHilsen () {
    alert("Hei!")
  }
  function oppstart(){

  }

Vi ber systemet om å utføre funksjonen (kalle opp funksjonen) ved å skrive

funksjonsnavnet etterfulgt av to parenteser, på samme måte som da vi definerte funksjonen

    function skrivUtHilsen() {
      alert("Hei!");
    }
    function oppstart() {
      skrivUtHilsen();
    }

Vi kan kalle funksjonen hvor mange ganger vi måte Ønske, og den blir utført en

gang for hvert kall.

      function skrivUtHilsen(){
      }
      function oppstart(){
        skrivUtHilsen();
        skrivUtHilsen();
        skrivUtHilsen();
      }

3.funksjon som verdi

En spesiell egenskap ved funksjoner i javaScript er at de også kan opptre som en

verdi på samme måte som et heltall eller en tekststreng. vi kan f.eks. lagre referanser

til funksjoner i en variabel.

    function summer(tall1, tall2){
      return tall1 + tall2;
    }
    function oppstart(){
      var mattefunksjon(3, 6) // 9
    }

I eksempelet over er effekten kun at vi lagrer funksjonen under et midlertidig navn.

Denne teknikken benyttes flittig når vi kobler hendelser til elementer. I en variabel/

egenskap lagres en referanse til funksjonensom skal benyttes, slik som:

    document.getElementById("btnUtfeor2).onclick = utfoer;

når vi benytter funksjoner som en verdi, har vi ikke med parenteser eller

argumenter. Vi benytter kun selve navnet. Det er fordi vi ikke ønsker å kalle

funksjonen, men referere til den.

Det er også mulig å lage en array av funksjon:>

function summer (tall1, tall2) {
           return tall1 + tall2;
  }
  function multipliser(tall1, tall2) {
    return tall1 * tall2;
  }
  function oppstart() {
    var mattefunksjon = [summer, multipliser];
    for(var i = 0; i < mattefunksjoner.lenght: i++) {
      alert(mattefunksjoner[i](3, 5)); // 8 i første meldign, 15 i neste
    }
  }
